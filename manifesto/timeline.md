# The Compressed Timeline of Computing

*Color: Gradient from #F0FDF4 (1950) to #052E16 (2024)*

---

## The Seven Decades of Compression

### 1950s - The Void Emerges (#052E16)
**The Primordial Era**

- **1950**: Alan Turing's "Computing Machinery and Intelligence"
- **1951**: UNIVAC I - First commercial computer
- **1952**: Grace Hopper invents the compiler
- **1956**: FORTRAN begins - First high-level language
- **1958**: LISP - Functions as first-class citizens

**Compression**: Room-sized → Building-sized computers
**Survived**: The stored program concept
**Lost**: Decimal computing (binary won)

---

### 1960s - Roots Take Hold (#14532D)
**The Language Explosion**

- **1960**: COBOL - Business speaks to machines
- **1964**: BASIC - Everyone can code
- **1967**: Simula - Objects are born
- **1969**: UNIX - Simplicity wins
- **1969**: ARPANET - 4 nodes connected

**Compression**: Mainframe → Minicomputer
**Survived**: Time-sharing, text editors, regular expressions
**Lost**: Punch cards (mostly)

---

### 1970s - The Forest Floor (#16A34A)
**The Structural Revolution**

- **1972**: C - The eternal language
- **1973**: Ethernet - Local networks
- **1975**: Microsoft founded
- **1976**: Apple I - Personal computing begins
- **1978**: K&R C "The C Programming Language"

**Compression**: Minicomputer → Microcomputer
**Survived**: C, UNIX philosophy, pipes
**Lost**: Goto considered harmful

---

### 1980s - Growth Accelerates (#22C55E)
**The Personal Era**

- **1981**: IBM PC - Standards emerge
- **1983**: C++ - Objects with performance
- **1985**: Windows 1.0
- **1987**: Perl - Duct tape of the internet
- **1989**: World Wide Web conceived

**Compression**: Desktop → Laptop
**Survived**: Object-orientation, GUIs, networking
**Lost**: 8-bit computing (mostly)

---

### 1990s - The Canopy Forms (#34D399)
**The Connection Era**

- **1991**: Linux - Open source wins
- **1993**: Mosaic browser
- **1995**: JavaScript - 10 days to change the world
- **1995**: Java - Write once, run anywhere
- **1999**: Y2K - Survival of COBOL proven

**Compression**: Desktop → Internet
**Survived**: Open source, web standards, scripting
**Lost**: Proprietary protocols

---

### 2000s - Deep Green (#4ADE80)
**The Scale Era**

- **2001**: Wikipedia - Crowdsourced knowledge
- **2003**: MapReduce paper
- **2004**: Facebook - Social graphs
- **2006**: AWS - Cloud computing
- **2009**: Bitcoin - Distributed consensus

**Compression**: Servers → Cloud
**Survived**: Distributed systems, NoSQL, REST
**Lost**: Physical media

---

### 2010s - Peak Verdant (#6EE7B7)
**The Mobile Era**

- **2010**: iPad - Post-PC era
- **2011**: Node.js - JavaScript everywhere
- **2014**: Kubernetes - Container orchestration
- **2015**: TensorFlow - ML democratized
- **2019**: GPT-2 - Language models scale

**Compression**: Apps → Services
**Survived**: Containers, microservices, ML
**Lost**: Monoliths (fighting back)

---

### 2020s - The Primary Green (#A7F3D0)
**The AI Era**

- **2020**: GPT-3 - 175B parameters
- **2021**: GitHub Copilot - AI pair programming
- **2022**: Stable Diffusion - Open source AI art
- **2023**: ChatGPT - Conversational AI
- **2024**: Sisu Repository - 70 years → 47KB

**Compression**: Code → Intention
**Surviving**: Prompts, fine-tuning, attention
**Losing**: Manual coding (but not really)

## The Patterns That Endure

### Constants Across All Decades:
1. **Complexity increases** → **Abstraction compresses it**
2. **Hardware improves** → **Software expands to fill it**
3. **New paradigm appears** → **Previous paradigm becomes a layer**
4. **Distribution increases** → **Synchronization becomes harder**
5. **Access democratizes** → **Quality varies wildly**

### The Compression Curve

```
1950s: 1 computer = 1 building
1960s: 1 computer = 1 room
1970s: 1 computer = 1 desk
1980s: 1 computer = 1 lap
1990s: 1 computer = 1 pocket
2000s: N computers = 1 cloud
2010s: ∞ computers = 1 service
2020s: All computers = 1 AI
2030s: ? = 1 thought
```

## The Color of Progress

Notice how our timeline moves from dark (void) to light (whisper)?

This is intentional. Early computing was close to the metal, dark and essential (#052E16). Modern computing is abstract, light and ethereal (#F0FDF4).

But Sisu argues: **The deepest green is the most valuable.**

The void (#052E16) contains everything. The whisper (#F0FDF4) contains almost nothing.
