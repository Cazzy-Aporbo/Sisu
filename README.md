# Sisu

**Sisu** is a deliberately minimal yet deeply annotated project.  
It compresses **70 years of computer-science history into ~47 kB of source code**.  
It is themed on the Finnish concept *sisu*: **stoic endurance, resilience, and calm determination against impossible odds.**

This repository is both **technical** and **aesthetic**: a miniature archive, a design object, and a curated demonstration of endurance in code.
