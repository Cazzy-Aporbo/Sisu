# Getting Started with Sisu

*Welcome to the compression of 70 years of computer science*

---

## Quick Setup

```bash
# Clone the repository
git clone https://github.com/[your-username]/sisu.git
cd sisu

# Run auto-configuration
chmod +x configure
./configure

# Build the timeline
make timeline

# Start breathing (optional but recommended)
make breathe
```

## Repository Structure

Each directory has a color representing its depth in the system:

- **`core/`** (#052E16 - Void Green): The deepest, most essential code
- **`paradigms/`** (#16A34A - Ancient Green): Different programming philosophies
- **`compression/`** (#4ADE80 - Deep Green): Algorithms that reduce complexity
- **`visualization/`** (#A7F3D0 - Primary Green): Visual representations
- **`manifesto/`** (#F0FDF4 - Whisper Green): Philosophy and documentation

## The Four Learning Paths

### 1. The Archaeologist 🏺
**Start**: `paths/archaeologist/journey.md`  
**For**: Those who want to understand from the metal up

You'll descend through layers of abstraction, from modern quantum computing down to raw assembly. Each file you encounter is a fossil from a different era.

### 2. The Compressor 🗜️
**Start**: `paths/compressor/journey.md`  
**For**: Minimalists who believe less is more

Learn how to reduce any algorithm to its essence. Study patterns that have survived because they cannot be simplified further.

### 3. The Visualist 🎨
**Start**: `paths/visualist/journey.md`  
**For**: Those who think in colors and patterns

Explore how gradients encode information, how color represents complexity, and how to visualize the invisible.

### 4. The Philosopher 🤔
**Start**: `paths/philosopher/journey.md`  
**For**: Those who ask "why" before "how"

Examine the ideas behind the code. From Turing's halting problem to modern compression theory.

## File Header Template

Every file follows this format:

```c
/*
 * =========================================================
 * SISU — [Component Name]
 * =========================================================
 * 
 * DENSITY:     ████████░░  [8/10]
 * ENDURANCE:   ██████████  [10/10]  
 * ABSTRACTION: ██████░░░░  [6/10]
 *
 * COLOR PHASE: #[HEX] ([Color Name])
 *
 * Historical Context:
 *   Born from: [Origin, Year]
 *   Survived:  [What it outlasted]
 *   Evolves:   [What it becomes]
 *
 * Compression Ratio: [Original]:[Current]
 *
 * =========================================================
 */
```
