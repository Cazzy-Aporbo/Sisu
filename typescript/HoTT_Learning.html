What is Homotopy Type Theory?

Homotopy Type Theory (HoTT) is a revolutionary field that connects type theory

Type Theory: A formal system where every value has a type, used as a foundation for mathematics and programming languages.
with homotopy theory
Homotopy Theory: A branch of topology that studies spaces up to continuous deformation.
. It proposes that types can be interpreted as topological spaces, and equalities between elements as paths in those spaces.

Simple Analogy: Types as Spaces

Think of a type as a space where each point is a value of that type. Two values being equal corresponds to a path between points. Higher equalities (equalities between equalities) correspond to higher-dimensional paths.

Why is HoTT Important?

HoTT provides a new foundation for mathematics that is particularly well-suited for computer formalization. It also offers new insights into the nature of equality and mathematical structures.

Core Concepts

HoTT introduces several groundbreaking ideas that reshape how we think about types, equality, and mathematical structures.

Identity Types
Identity Types: In type theory, the type of proofs that two elements are equal. In HoTT, these correspond to paths.
as Paths

In HoTT, a proof that a = b is interpreted as a path from point a to point b in the space represented by the type. This geometric interpretation provides intuitive reasoning about equality.

// In type theory, identity types are fundamental
Id : A → A → Type
// For any x : A, refl(x) : Id(x, x) is the reflexive path

Univalence Axiom
Univalence Axiom: States that equivalent types are equal. This is a profound connection between logical equivalence and equality.

The Univalence Axiom is perhaps the most revolutionary aspect of HoTT. It states that if two types are equivalent (can be transformed into each other without loss of information), then they are actually equal.

What does Univalence mean in practice?
It means that if we have two types A and B, and we can prove they're equivalent (there's a bijection between them), then we can treat A and B as literally the same type. This allows us to transfer properties and constructions between equivalent types.

Higher Inductive Types
Higher Inductive Types: Types defined by not only their points but also paths and higher paths between these points.

Traditional inductive types define types by specifying how to construct their elements. Higher inductive types also allow specifying paths between elements, enabling the direct definition of spaces with non-trivial topology.

// Example: The circle as a higher inductive type
data S¹ : Type where
  base : S¹
  loop : Id(base, base)

Type = Space

The fundamental insight of HoTT is the correspondence between type theory and homotopy theory, often summarized as "types are spaces".

Interpretation Dictionary

HoTT establishes a precise correspondence between type-theoretic concepts and homotopy-theoretic concepts:

/* Type Theory */    /* Homotopy Theory */
Type            ↔    Space
Term            ↔    Point
Identity type    ↔    Path space
p : Id(a,b)    ↔    Path from a to b
Id(p,q)      ↔    Homotopy between paths

Implications for Mathematics

This correspondence allows mathematicians to use geometric intuition when working with types, and computational tools when working with spaces. It has already led to new insights in both fields.
