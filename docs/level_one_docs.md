# Level One Assembly Tutorial Documentation

## Overview
This repository contains foundational assembly programs designed to teach:
- Basic loops and recursion
- Array and string manipulation
- Simple data structures (linked lists)
- Basic system calls for Linux

The examples focus on clean, readable, and human-teachable code.

## Examples Included
| Example | Concept | Notes / Variants |
|---------|---------|-----------------|
| Sum Array | Looping over array elements | Iterative and unrolled versions |
| Factorial | Recursion | Recursive and iterative versions |
| Fibonacci | Sequence generation | Iterative and recursive versions |
| String Length | Pointer arithmetic | Null-terminated strings |
| String Copy | Memory movement | Byte-wise copy |
| Matrix Multiply | Nested loops | 3x3 matrix multiplication |
| Linked List | Pointers | Insert and traverse |
| Bubble Sort | Sorting | Standard algorithm |
| File I/O | Syscalls | Read and write example |
| Simple Calculator | Arithmetic | +, -, *, / operations |
