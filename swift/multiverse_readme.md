HOW TO PLAY
1. Paste into a macOS → Xcode → File ▸ New ▸ Playground ▸ Blank.
2. Show the Debug Area (⇧⌘Y).
3. Run.
4. In the console type three integers separated by spaces: universeIndex row col then ↩.
Example: 3 20 60 pokes universe 3 at cell (20, 60) and opens a wormhole.
5. Watch the checksum flicker like a Buddhist sand mandala.
6. When you are ready to let go, simply stop the playground—an entire multiverse vanishes, but the poem remains.
